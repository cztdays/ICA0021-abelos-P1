# Teise tudengi hindamise juhend

- Klooni repo ja loo virtuaalkeskkond.
- Käivita `python -m pytest` (või vastav testkäsk).
- Käivita demo dry-run: `python src/__main__.py --config config.example.yaml --dry-run`
- Tee issue kui leiad vea, või simple PR parandusega (branch: fix/<issue>).
- Reviewer kirjeldab run-logis, mis testid jooksid (attach).
